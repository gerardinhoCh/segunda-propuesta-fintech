# de Shop en Wall Street — Inversión Inteligente en ETFs y Mercados Regulados

Landing page profesional para educación financiera e inversión en mercados regulados. Construida con **Next.js 16**, **React 19**, **Tailwind CSS 4** y **Framer Motion**.

## 🚀 Características

- **Diseño fintech premium** con glassmorphism, gradientes animados y micro-interacciones
- **Calculadora de interés compuesto** interactiva con gráficos SVG en tiempo real
- **Portafolio verificado** con resultados reales de inversión en ETFs
- **SEO optimizado** con meta tags, Open Graph, Twitter Cards, sitemap dinámico y robots.txt
- **Responsive** — diseño adaptado a mobile, tablet y desktop
- **Dark mode** con paleta de colores fintech profesional

## 🛠️ Instalación

```bash
# Clonar el repositorio
git clone https://github.com/gerardinhoCh/segunda-propuesta-fintech.git

# Instalar dependencias
npm install

# Ejecutar en desarrollo
npm run dev

# Construir para producción
npm run build
```
